# Get Printing Us API

## Setup a .env file before start

Variables that the file must be have

```
PORT=
TOKEN_KEY=
DB_HOST=
EMAIL_USER=
EMAIL_PASS=

```

## Setup NPM

```
npm install
```

## Development

```
npm run dev
```

## Setup YARN

```
yarn install
```

## Development

```
yarn run dev
```
